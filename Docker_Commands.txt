# Docker Commands

# Here are some useful Docker commands for working with this project:

# Stop the container:

docker stop my-blog-app

# Remove the container:

docker rm my-blog-app

# Remove the container image:

docker rmi my-blog-app

# Docker Compose

# Alternatively, you can use Docker Compose to simplify container m# anagement. Ensure you have Docker Compose installed (refer to the# prerequisites). Then, run the following command:

docker-compose up -d

# To stop and remove the containers created by Docker Compose, use:

docker-compose down
